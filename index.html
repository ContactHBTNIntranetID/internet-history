OVERVIEW OF THE WORLD WIDE WEB

The world wide web, often simply referred to as the Web, is a vast system of interconnected digital information and resources that are accessible via the internet. It is one of the most significant and transformative inventions of the 20th century, revolutionizing how people access, share and interact with information globally. CONTINUATION OF WEB OVERVIEW

1. Invention and Evolution:

The World Wide Web was invented by the British computer scientist Sir Tim Berners-Lee in 1989, while working at CERN (European Organization for Nuclear Research) in Switzerland. He proposed the idea of using a system of hypertext documents, connected by hyperlinks, to facilitate information sharing and collaboration among researchers.

2. Basic Components:

• Web Pages: Web pages are documents containing text, images, videos and other multimedia elements, formatted using languages like HTML (Hypertext Markup Language).

• Hyperlinks: Hyperlinks, also known as links, are clickable elements that allow users to navigate between different web pages, websites or online resources.

• URLs: Uniform Resource Locators are unique web addresses used to identify and locate specific web pages or resources on the internet.

• Web Browsers: Web browsers are software applications that allow users to access and view web pages. Examples include Chrome, Firefox, Safari and Edge.

3. Client-Server Model:

The Web operates on a client-server model. Users(clients) request web pages from web servers, which host and deliver the requested content.

4. Hypertext Transfer Protocol (HTTP):

HTTP is the protocol used for transferring data between a web browser and a web server. It defines how requests and responses should be formatted and processed.

5. Hypertext Markup Language (HTML):

HTML is the standard markup language used to create the structure and content of web pages. It includes elements for text, images, links and other multimedia.

6. Web Development Technologies:

In addition to HTML, web development involves other technologies like CSS (Cascading Style Sheets) for styling and layout, JavaScript for interactivity and dynamic content, and more recently, server-side languages and frameworks.

7. Search Engines:

Search engines like Google, Bing and Yahoo index organize vast amounts of web content, enabling users to search for and find specific information easily.

8. Social Media and User-Generated Content:

Web has evolved to include social media platform (e.g., Facebook, Twitter, Instagram) where users can create, share and interact with content.

9. E-commerce and Online Services:

Web has become a hub for e-commerce, enabling online shopping, digital transactions and various online services (e.g., banking, streaming, education).

10. Mobile and Responsive Design:

the rise of mobile devices, responsive web design ensures that web pages adapt to different screen sizes and orientations.

11. Web Standards and Accessibility:

like the World Wide Web Consortium(W3C) establish web standards to ensure interoperability and accessibility across different devices and users, including those with disabilities.

12. Challenges:

The Web has introduced challenges such as information overload, data privacy concerns, cybersecurity threats and the spread of misinformation.

13. Future Trends:

The Web continues to evolve with emerging technologies like artificial intelligence, virtual reality and blockchain, shaping how information is accessed and shared in the future. Overall, the World Wide Web has profoundly transformed communication, information dissemination, and various aspects of modern life, connecting people and resources on a global scale.

1.2 History of the Internet

The internet is at once a world-wide broadcasting capability, a mechanism for information dissemination, and a medium for collaboration and interaction between individuals and their computers without regard for geographic location. The internet represents one of the most successful examples of the benefits of sustained investments and commitment to research and development of information infrastructure. Beginning with the early research in packet switching, the government, industry and academia have been partners in evolving and deploying this exciting new technology. Today, terms like bleiner@computer.org and https://www.acm.org trip lightly off the tongue of the random person on the street. The history of the internet revolves around four distinct aspects. There is the technological evolution that began on early research on packet switching and the ARPANET (and related technologies), and where current research continues to expand the horizons of the infrastructure along several dimensions, such as scale, performance and higher-level functionality. There is the operations and management aspect of a global and complex operational infrastructure. There is the social aspect, which resulted in a broad community of internauts working together to create, and evolve the technology. And there is commercialization aspect, resulting in an extremely effective transition of research results into a broadly deployed and available information infrastructure. The internet today is a widespread information infrastructure, the initial prototype of what is often called the National (or Global or Galactic) information infrastructure. Its history is complex and involves many aspects – technological, organizational and community.






1.3 WWW VS THE INTERNET

The Web and the internet are closely related but distinct concepts. The internet is a global network of interconnected computers and servers that communicates using various protocols. It includes emails, file transfer and other forms of data exchange. On the other hand, the Web is a system of interconnected documents and resources, accessed through the internet. It’s a subset of the internet that allows users to view and interact with web pages, navigate through hyperlinks, and access multimedia content. In simpler terms, the internet is the infrastructure that enables various services, while the World Wide Web is one of the most prominent services running on that infrastructure. Internet: The Internet is a vast global network of interconnected computer networks. It's a network of networks, connecting millions of computers, servers, devices, and data centres around the world. The Internet enables communication and data transfer among these connected devices using various protocols, including TCP/IP (Transmission Control Protocol/Internet Protocol). The Internet existed before the World Wide Web and encompasses a wide range of services and technologies beyond just web browsing. It includes email, instant messaging, online gaming, file sharing, remote access, and more. World Wide Web (WWW): The World Wide Web, often referred to as the "Web," is a specific subset of services and content available on the Internet. It's a system of interlinked hypertext documents and multimedia content that can be accessed via the Internet. The Web is built on top of the Internet's infrastructure and relies on technologies such as HTTP (Hypertext Transfer Protocol) and HTML (Hypertext Markup Language) to display and link content. When you use a web browser to access websites, read articles, watch videos, and interact with web-based applications, you're using the World Wide Web. Websites are hosted on servers connected to the Internet, and you access them by typing in URLs (Uniform Resource Locators) or clicking on links. In summary, the Internet is the underlying global network that enables the connectivity of devices and the exchange of data, while the World Wide Web is a subset of services on the Internet that allows users to access and interact with multimedia content and information through web browsers. The Web is just one of many services that the Internet facilitates.



1.4 EVOLUTION OF THE INTERNET

Each phase of the internet's evolution has brought about significant changes in how people communicate, access information and conduct various aspects of their lives.

1. Foundation and Early Networking (1960s-1970s):

The internet's origins lie in research projects like ARPANET, which connected computers for scientific and military purposes. The development of key protocols like TCIP/IP in the 1970s established the framework for data exchange across interconnected networks.

2. Emergence of the World Wide Web (1980s-1990s):

Tim Berners-Lee's invention of the World Wide Web in 1989 allowed the creation of easily accessible web pages using hyperlinks. This marked the shift from simple data exchange to user-friendly browsing and interaction.

3. Commercialization and Dot-com Boom (1990s-2000s):

The 1990s saw the commercialization of the internet, leading to the dot-com boom and the rise of the e-commerce, search engines, and early social media platforms.

4. Web 2.0 and social media (2000s):

The term “Web 2.0” emerged to describe a more interactive and user-driven internet, characterized by programs like Facebook, YouTube and Wikipedia. Social media and user-generated content became central.

5. Mobile and Connectivity(2010s):

With the proliferation of smartphones and mobile devices, the internet became more accessible and pervasive. Mobile apps, streaming services, and the expansion of wireless connectivity reshaped how people use the internet.

6. Internet of Things (IoT) and Beyond (2010s- Present):

The IoT emerged, connecting everyday objects to the internet, enabling data exchange and automation on a larger scale. Concepts like cloud computing, big data, and artificial intelligence have gained prominence, transforming industries and the way data is processed.

7. Focus on Privacy and Security (ongoing):

Growing concerns about privacy, cybersecurity, and data protection have led to increased efforts to secure online communication, regulate data usage, and ensure user privacy.

8. Future Developments:

The internet continues to evolve with advancements in technologies like 5G, augmented reality, virtual reality, and decentralized technologies like blockchain. The evolution is likely to be shaped by the interplay of technological innovation, regulatory measures and societal needs.

INTERNET PROTOCOLS:

Internet protocols are standardized rules that define how data is transmitted, received, and processed across computer networks. They enable different devices and systems to communicate effectively.

1. Transmission Control Protocol (TCP): TCP is a connection-oriented protocol responsible for ensuring reliable and ordered data delivery between devices. It breaks data into packets, assigns sequence numbers to them, and manages acknowledgements and retransmissions to ensure data integrity.
2. Internet Protocol (IP): IP is a fundamental protocol that handles routing and addressing. It provides logical addressing for devices connected to the internet, allowing data packets to be routed from source to destination across different networks.
3. Hypertext Transfer Protocol (HTP): HTTP is the foundation of data communication on the World Wide Web. It defines how web browsers and web servers interact. HTTP facilitates the request and transfer of web pages, images, videos and other resources.

4. Hypertext Transfer Protocol Secure (HTTP): HTTP is a secure version of HTTP that uses encryption (usually provided by SSL/TLS protocols) to protect the privacy and integrity of data transmitted between users and websites. It ensures secure online transactions and sensitive information sharing. 5. File Transfer Protocol (FTP): FTP is used for transferring files between a client and a server over a network. It provides functionality to upload, download, delete and manages files on remote servers. 6. Simple Mail Transfer Protocol (SMTP): SMTP is the protocol used for sending email messages from a client to a server or between servers. It defines how emails are relayed, forwarded and delivered to recipients’ mail servers. 7. Post Office Protocol version 3 (POP3) and Internet Message Access Protocol (IMAP): These are emails retrieval protocols. POP3 downloads emails to a local device, while IMAP allows users to access and manage their emails directly on the email server, keeping messages synchronized across multiple devices. These protocols, along with many others, work together to enable seamless communication and data exchange across the internet, supporting a wide range of applications and services that we use in our daily lives.

WEB ARCHITECTURE

Web application architecture underscores the systematic design and configuration of web applications. This intricate structure generally comprises three principal constituents: the client-side, server-side, and the database. Contemporary web applications commonly adopt architectural patterns like Model-View-Controller (MVC) or analogous paradigms to segregate responsibilities and augment the sustainability of the system.

Client-side and server-side engineering:

Client-side engineering embodies the realm of developing user interface components and user experience elements of a web application, all of which operate within the user's web browser. This encompasses the formulation of HTML, CSS, and JavaScript programming components. Conversely, server-side engineering encompasses the creation of the backend framework of the web application. This facet is responsible for executing data processing, orchestrating business logic, and orchestrating interactions with databases, thereby underpinning the application's foundational functionality
